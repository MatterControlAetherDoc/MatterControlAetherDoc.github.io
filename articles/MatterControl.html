MatterControl

Overview

MatterControl was originally developed by MatterHackers as a GUI interface to their own slicer, MatterSlice. A modified version of MatterSlice has been created to accept Aether specific tool configurations and to generate G-Code that is accepted by the Aether 1 bioprinter.

Similarly, MatterControl itself has been modified to allow for GUI specification of tools and to allow the GUI to appropriately tell the slicing engine about the tools and their associated settings.

The GUI is not generally responsible for generating custom G-Code for the Aether 1. The only G-Code generated by the GUI is the startGCode which is generated to contain the extruder heating commands.

Tool Storage And Integration

ToolSettings

Unlike the additions to MatterSlice, the GUI modifications were not implemented with object oriented design. All possible tool settings of all possible tools are aggregated into one ToolSettings class in the file SlicerConfiguration/Settings/ToolSettings.cs. Property names in the ToolSettings class must have names that correspond to tool settings in the new MatterSlice. The class PrinterSettings was modified to contain a list of ToolSettings objects that are used to store tool configuration data. The current tools and their settings may be accessed globally through one of the ActiveSliceSettings static class properties, i.e., ActiveSliceSettings.Instance.Tools.

The tool settings stored in ActiveSliceSettings.Instance.Tools are used in the generation of a configuration file to feed to the slicing engine. The tool settings export code in located in the MappedTools class in the file SlicerConfiguration/SlicerMapping/EngineMappingMatterSlice.cs. This class is responsible for generating the lines of the configuration file relating to tool settings. This is done by iterating over all the ToolSettings instances in ActiveSliceSettings.Instance.Tools and reading all of the non-zero properties to generate a toolsSettings string as discussed in the MatterSlice article.

ToolSettingsWidget

The GUI widget used to display and configure tool settings is defined in the file SlicerConfiguration/ToolSettingsWidget.cs. This GUI widget is added as a child in the SliceSettingsWidget in the CreateSideTabsAndPages(OrganizerCategory) instance method.
